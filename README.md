# Jaroor Fitness Website

A modern, responsive website for Jaroor Fitness Gym in Bhilai, Chhattisgarh.

## Features

### 🎨 **Design & UI**
- Modern, clean design with fitness-focused aesthetics
- Responsive layout that works on all devices
- Smooth animations and hover effects
- Professional color scheme with orange and dark blue accents

### 📱 **Responsive Design**
- Mobile-first approach
- Fully responsive navigation with hamburger menu
- Optimized layouts for tablets and desktop
- Touch-friendly interactions

### 🔧 **Functionality**
- Smooth scrolling navigation
- Mobile hamburger menu
- Contact form with validation
- Back-to-top button
- Scroll animations
- Auto-rotating testimonials
- Interactive gallery

## Installation & Usage

### Local Development
1. Download all files to a folder
2. Open `index.html` in any modern web browser
3. No server required for basic functionality

### Web Hosting
1. Upload all files to your web hosting provider
2. Ensure the file structure is maintained
3. Access the website via your domain

## Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #ff6b35;    /* Main accent color */
    --secondary-color: #2c3e50;  /* Dark text color */
    --accent-color: #e74c3c;     /* Secondary accent */
}
```

### Content
Update text content in `index.html`:
- Gym information
- Service descriptions
- Trainer profiles
- Contact details
- Testimonials

### Images
Replace placeholder images with actual gym photos:
- Hero background
- Gallery images
- Trainer photos
- Feature icons
